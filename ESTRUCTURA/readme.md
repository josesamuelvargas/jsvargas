### prueba

 - index.html
 - public

en el archivo html la etiqueta `title`establece el titulo de la pagina.

```css
p {
    color: rgb(0, 195, 255);
    font-style: italic;
}
.uno .tres{
    background-color: red;
  }
  .uno>.tres{
    font-size: 50px;
  }
  
  .dos{
    font-style: italic;
    text-align: center;
  }
```
